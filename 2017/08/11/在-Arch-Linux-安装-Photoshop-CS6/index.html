在 Arch Linux 安装 Photoshop CS6

在 Linux 中用 Photoshop 非常不容易，因为 Adobe 没有官方支持的 Linux 版本，所以为了使用 Photoshop 只能在虚拟机中或者使用 wine，本文主要介绍在 Arch Linux 中使用 wine 安装 Photoshop 的方法。

准备工作

使用 pacman 安装 wine 和 PlayOnLinux

配置 wine

  1. 打开安装好的 PlayOnLinux 选择左边的 Install a program
  2. 等加载的圈圈转完选左下角的 Install a non-listed program
  3. 弹出手动安装的界面点击 Next 直到提示问你要怎么做的选项界面，选择 Install a program in a new virtual drive 点击 Next
  4. 在输入框输入 Photoshop 点击 Next
  5. 多选 Configure Wine 和 Install some libraries
