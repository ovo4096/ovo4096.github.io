在 Arch Linux 安装 Photoshop CS6

在 Linux 中用 Photoshop 非常不容易，因为 Adobe 没有官方支持的 Linux 版本，所以为了在 Linux 中使用 Photoshop 只有虚拟机和 wine 两种方案了。

Wine 比起虚拟机可以更少的占用系统资源，也可以更好的和其它本地软件交互，缺点是配置起来比较复杂，至少我没有一次安装成功，所以本文介绍在 Arch Linux 安装 Photoshop CS6 的方法。

准备工作

使用 pacman 安装 wine 和 PlayOnLinux。

这里可能有的同学会问为啥要用 PlayOnLinux，其实我只是想方便的管理 wine 版本和通过选项来安装 wine 依赖库（在看懂教程后用 winetricks 可能会更方便）。

还有本文重点不是通过 PlayOnLinux 安装 Photoshop ，因为 PlayOnLinux 提供的安装 Photoshop CS6 的脚本并不能正常工作在 Arch Linux，所以重点是利用 pacman 安装的 Wine 版本来安装 Photoshop。

从 官网 下载 Photoshop CS6 的安装程序，具体位置在网页下方的 Photoshop CS6, Photoshop CS6 Extended 选项当中，最好选择 English 版本其它版本我没测试过。

完成上面两步骤，准备工作就完成了。

配置 wine

打开 PlayOnLinux 点击左边栏中的 Install a program 之后在新界面左下角选择 Install a non-listed program 打开手动安装向导。

点击 Next。

选择 Install a program in a new virtual drive 点击 Next。

填写这个新的虚拟驱动器名称，因为是给 Photoshop 用的，所以这里填 Photoshop 或者你认为合理的名字就好，点击 Next。

勾选 Configure Wine 和 Install some libraries，点击 Next。

选择 32 bits windows installation，点击 Next。

在配置 wine 的界面，确保 Windows Version 为 Windows 7，点击 OK。

勾选 wine 需要的依赖组件：

  • POL_Function_FontsSmoothBGR
  • POL_Function_FontsSmoothGrayScale
  • POL_Function_FontsSmoothRGB
  • POL_Install_atmlib
  • POL_Install_corefonts
  • POL_Install_FontsSmoothBGR
  • POL_Install_FontsSmoothGrayScale
  • POL_Install_FontsSmoothRGB
  • POL_Install_gdiplus
  • POL_Install_gecko
  • POL_Install_ie8
  • POL_Install_msxml3
  • POL_Install_msxml6
  • POL_Install_tahoma
  • POL_Install_tahoma2
  • POL_Install_vcrun2005
  • POL_Install_vcrun2008

点击 Next 接下来只要看到 I Agree 选项都勾选并点击 Next 这一步骤可能要重复多次。

期间需要配置 ie8 的语言版本选择 English。

完成上面的步骤，应该可以看到选择安装程序的界面了，解压从官网下载的 Photoshop CS6 安装程序包，点击 Browse 选择解压目录中的 Adobe CS6/Set-up.exe 点击 Next。

如果可以看到 Photoshop 安装界面，那么说明你的 Wine 配置没有问题了，接下来需要安装 Photoshop 本体了。

安装 Photoshop CS6

接着上面步骤来安装 Photoshop 本体，选择 Try 进入 Adobe 软件许可界面。

点击 Accept。

点击 Sign in 因为 Wine 版的 Photoshop 无法登录 Adobe 帐号，所以这里需要等它网络请求超时，通过离线的方式安装。

点击 Connect Later。

如果不需要特别其它配置，点击 Install 等待安装完成。

点击 Close，完成安装。

选择 Photoshop.exe 创建 Wine 运行的快捷方式。

选择 I don’t want to make anther shortcut 点击 Next 完成全部的安装。

做到这一步 Photoshop 应该成功安装了😂，尝试在 PlayOnLinux 或者桌面图标运行它。

但是好像仍然不能打开它，原来是 Photoshop 要求我们登录 Adobe 帐号但是不管怎么点 Sign in 都无法登录，所以我们只能通过破解 Photoshop 的方式绕过登录了（只能怪 Adobe 不支持 Linux 还这么毒瘤）。。。

破解 Photoshop

破解非常简单，先关闭之前打开的 Photoshop， 用 PlayOnLinux 或者自己找到 Photoshop 的安装目录，替换 amtlib.dll 为 破解版的 amtlib.dll，然后再次运行。

好的，看到上面的界面说明你的 Photoshop 已经成功安装并且可以正常运行了 😂😂😂。

常见问题

如果打开 PSD 文件，软件内的编辑区域是黑屏或者花屏的，在 Photoshop 设置中设置不使用图形处理器，然后重启它再试。
